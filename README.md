# پروژه پلتفرم فین‌تک - Iran Credit

این پروژه، پیاده‌سازیِ فرانت‌اندِ یک پلتفرمِ بین‌المللیِ پرداخت و صرافی ارز دیجیتال است که با هدفِ ایجادِ تجربه‌ی کاربریِ روان و پرسرعت طراحی شده است.

## معماری و ساختار پروژه

این پروژه در یک مخزن واحد ادغام شده است تا ساختارِ ماژولار و تمیزِ کدنویسی را به نمایش بگذارد. صفحات پروژه عبارتند از:

* **درباره ما (About Us):** معرفیِ اهداف و استراتژی‌های پلتفرم.
* **بلاگ (Blog):** بخشِ مقالات با طراحیِ ریسپانسیو.
* **سرویس‌ها (Service):** نمایشِ سرویس‌هایِ مالی و صرافی.
* **ارتباط با ما (Contact Us):** فرم‌هایِ ارتباطی با اعتبارسنجیِ ورودی‌ها.
* **مستندات (Documentation):** راهنمایِ فنیِ کاربران.

## تکنولوژی‌هایِ مورد استفاده

* **فرانت‌اند:** HTML5, Modern JavaScript (ES6+)
* **استایل‌دهی:** Tailwind CSS (برای ساختارِ ریسپانسیو و مدیریتِ کامپوننت‌ها)
* **فلسفه طراحی:** پیاده‌سازیِ دقیقِ طراحی‌هایِ فیگما (Figma) با تمرکز بر دقتِ پیکسل‌ها.

## ویژگی‌های کلیدی

* **طراحی واکنش‌گرا (Responsive Design):** بهینه‌سازیِ کامل برای تمامیِ اندازه‌هایِ صفحه (موبایل، تبلت، دسکتاپ).
* **کارایی (Performance):** استفاده از استراتژی‌هایِ بهینه‌سازی برای افزایشِ سرعتِ بارگذاری.
* **قابلیت نگهداری:** ساختارِ پوشه‌بندیِ منطقی برای توسعه‌هایِ آتی.

## نحوه اجرا

برای اجرایِ این پروژه در سیستمِ خود، کافیست مخزن را Clone کرده و پوشه اصلی را در VS Code باز کنید.
